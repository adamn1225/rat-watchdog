# RAT Blocker

A PowerShell-based solution for blocking Remote Access Trojans (RATs).

## Project Structure

- **core/** - Core functionality scripts
  - `enable.ps1` - Enable RAT blocking
  - `disable.ps1` - Disable RAT blocking
  - `status.ps1` - Check current status

- **gui/** - Graphical user interface
  - `rat-blocker.ps1` - Windows Forms GUI

- **deploy/** - Deployment scripts
  - `intune.ps1` - Microsoft Intune deployment
  - `gpo.ps1` - Group Policy deployment
  - `rmm.ps1` - RMM tool deployment

- **logging/** - Logging functionality
  - `agent.ps1` - Logging agent
